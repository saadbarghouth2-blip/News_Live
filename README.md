GNews - Ultra Professional UI

This is a skeleton project (React + Vite + Tailwind + Framer Motion) with an Express proxy server for GNews API.

Instructions:
1. Put your GNEWS_API_KEY into server/.env as GNEWS_API_KEY=YOUR_KEY
2. In one terminal: cd server && npm install && npm run dev
3. In another terminal: cd web && npm install && npm run dev
4. Open the web dev server (Vite) shown in the terminal.

NOTE: .env is intentionally excluded from the ZIP. Add it locally.
